# 🌲 Krafton Jungle Archives

> **From Bare Metal to Modern Software Engineering**

이 저장소는 **크래프톤 정글(Krafton Jungle)** 5개월간의 치열한 몰입 과정을 기록한 아카이브입니다.
전산학의 기초가 되는 **자료구조**와 \*\*운영체제(OS)\*\*를 밑바닥부터 구현하며 CS 지식을 내재화하였고, 이를 바탕으로 **Go(Golang)** 언어를 학습하여 실무 시장에서 경쟁력 있는 나만의 무기를 만들어가는 과정을 담았습니다.

## 🚀 The Journey: Growth & Philosophy

저의 개발 여정은 \*\*"왜?"\*\*라는 질문에서 시작해 **"어떻게"** 해결할지로 이어집니다.

### 1\. Deep Dive into CS (The Roots) 🛠️

  * **Low-Level 이해:** C언어를 통해 직접 메모리를 관리(`malloc`)하고, 복잡한 자료구조(`Red-Black Tree`)를 구현하며 포인터와 메모리 구조를 깊이 있게 이해했습니다.
  * **시스템의 핵심, OS:** KAIST PintOS 프로젝트를 통해 스레드 스케줄링, 가상 메모리(VM), 파일 시스템을 직접 구축하며 운영체제의 작동 원리를 체득했습니다.
  * **네트워크:** 소켓 프로그래밍을 통해 `Web Proxy` 서버를 구현하며 동시성(Concurrency)과 HTTP 프로토콜을 다뤘습니다.

### 2\. Specialization with Go (The Weapon) ⚔️

  * CS 지식을 바탕으로, 높은 생산성과 성능을 보장하는 현대적인 언어 **Go**를 최종 주특기로 선택했습니다.
  * 단순한 기능 구현을 넘어, 동시성 처리와 마이크로서비스 아키텍처 등 Go의 장점을 극대화한 프로젝트를 수행하며 개발자로서의 전문성을 확립했습니다.

## 📝 Retrospective

크래프톤 정글에서의 5개월은 \*\*'엔지니어링적 사고'\*\*를 기르는 시간이었습니다.
운영체제를 만들며 겪었던 수많은 커널 패닉과 디버깅 과정은 문제 해결에 대한 끈기를 길러주었고, Go 언어를 통한 프로젝트는 실무적인 감각을 깨우쳐 주었습니다.

이곳에 정리된 코드들은 제가 \*\*"어떤 문제를 만나도 깊이 있게 파고들어 해결할 수 있는 개발자"\*\*임을 증명하는 발자취입니다.
